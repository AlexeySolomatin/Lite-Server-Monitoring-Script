# Служебный модуль ядра (Core)

## Описание

Модуль `core` является базовым компонентом **Lite Server Monitor (LSM)**. Он отвечает за функционирование ключевых механизмов системы, включая работу фоновых системных служб и управление расписанием ежедневной сводной отчётности.

> **Важно:** Данный модуль отмечен флагом `MODULE_IS_CORE="true"` и является критически важным для работы LSM. Его не следует удалять или отключать.

## Назначение и функции

- **Ежедневная отчётность:** Автоматический вызов сбора данных со всех активных модулей LSM и отправка сводного отчёта о состоянии сервера.
- **Управление расписанием:** Регулярный запуск генератора отчётов с помощью таймера systemd.
- **Системные зависимости:** Обеспечение работоспособности базовых библиотек и единой структуры юнитов systemd.

## Состав модуля

- `lsm-report.service` — One-shot служба systemd, выполняющая генерацию и отправку отчёта.
- `lsm-report.timer` — Таймер systemd, вызывающий службу отчётов по расписанию.

## Устанавливаемые файлы

```
/etc/systemd/system/lsm-report.service
/etc/systemd/system/lsm-report.timer
/opt/lsm/modules/core/manifest.sh
```

## Расписание по умолчанию

Таймер lsm-report.timer запускает генерацию отчёта каждые сутки (по умолчанию в 08:00 или 00:00 в зависимости от конфигурации lsm-report.timer).
